## 概要
大学研究室内で食料品を販売するにあたり，主に以下のことを行うシステム．
* 食料品の情報表示（メニュー）
* 食料品の在庫管理
* 食料品の販売記録

## 動作環境
下記のいずれかの組み合わせを想定し開発．
* ver 1.0系
  * PHP 7.3
  * MySQL 5.5.62
* ver 2.0系
  * PHP 7.4
  * MariaDB 10.6
