# InventoryControl（開発中）
![GitHub](https://img.shields.io/github/license/penguin-syan/InventoryControl)

研究室内の食品販売用システム
## 概要
大学研究室内で食料品を販売するにあたり，以下のことを行うシステム．
* 食料品の情報表示（メニュー）
* 食料品の在庫管理
* 食料品の販売記録

## 開発環境・動作確認済み実行環境
* PHP 7.3
* mysql 5.5.62
